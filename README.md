##  Description

A habit tracker app built with React and Vite. Track your daily habits, build streaks, and visualize your progress.



## Features

- **CRUD Habits** â€” Add, edit, and delete habits with 6 categories (Health, Fitness, Productivity, Learning, Mindfulness, Social)
- **Daily Tracking** â€” Mark habits as done with checkboxes (today only to prevent cheating)
- **Streak Counter** â€” Auto-calculated streaks based on consecutive completed days
- **Dashboard** â€” Summary cards (today's habits, current streak, completion rate) and habit cards with progress bars
- **Calendar** â€” GitHub-style heatmap showing habit consistency; darker = more habits completed
- **Statistics** â€” Charts, progress bars, and analytics
- **Auth** â€” Simple login/signup (UI only, no backend)
- **Theme** â€” Light and dark mode with persistence
- **Responsive** â€” Works on mobile, tablet, and desktop
- **Data Persistence** â€” localStorage for habits and completions
